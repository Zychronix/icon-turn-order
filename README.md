# About
Icon Turn Order display is the standalone mod of the Icon Turn Order display originally featured in Dragoon Modifier. This is currently compatible with Severed Chains devbuild for Recommended Build 3 (RB3).

![Example turn order image](turnorder.png)

# Installation

Drag and drop it into the Severed Chains/mods folder. If a "mods" folder does not exist create one.

# Known Issues
The following monsters are missing from the turn order and will be displayed with a "?"

Crafty Thief, DD Ball, DD Cannon, Windigo Heart, Michael (core), Madman
